# PlayerActiveSkill 기획 문서

이 문서는 인게임 플레이어 액티브 스킬 시스템의 기획 내용을 정리합니다.
기존 로그라이크 스킬 선택 시스템(볼/패시브 스킬)과는 독립된 시스템이며,
플레이어가 버튼을 직접 눌러 발동하는 2종의 스킬로 구성됩니다.
현재 버전은 프로토타입이므로 스킬 선택 없이 게임 시작 시 2개를 고정 지급합니다.

---

## 1. 개요

| 항목 | 내용 |
|------|------|
| 시스템 성격 | 기존 볼/패시브 스킬 선택 시스템과 별개 |
| 발동 방식 | 인게임 UI 버튼을 플레이어가 직접 탭하여 발동 |
| 스킬 지급 방식 | 프로토타입 기준: 게임 시작 시 2종 전부 고정 지급 |
| 수치 관리 | 모든 수치는 ScriptableObject로 Inspector에서 수정 가능 |

---

## 2. 스킬 목록

| 스킬명 | 쿨타임 | 효과 | 수치 |
|--------|--------|------|------|
| 버서크 | 30초 | 지속 시간 동안 모든 활성 볼 속도 증가 | 속도 1.5배, 지속 6초 |
| 분신 | 30초 | 원본 로스터 볼을 복사하여 순차 발사 | 복사본은 2회 회수 시 소멸 |

---

## 3. 공통 규칙

- 테스트 단계에서는 게임 시작 시 2개 모두 즉시 사용 가능
- 쿨타임 시작 시점: 스킬 발동 즉시
- 볼이 비행 중인 상태에서도 언제든 사용 가능
- 초기 발사와 회수 후 재발사는 동일한 간격으로 순차 처리
- 회수된 볼은 선입선출(FIFO) 순서로 재발사
- 모든 순차 발사는 실제 발사 시점의 현재 조준 궤도를 사용

---

## 4. 스킬 상세

### 4-1. 버서크

- 발동 시점에 씬 내 모든 활성 볼의 이동 속도를 1.5배로 증가
- 6초 동안 유지 후 원래 속도로 복귀
- 지속 시간 중 새로 발사되거나 재발사되는 볼과 분신에도 같은 배율 적용
- 활성 중 캐릭터 이펙트는 에셋 제작 단계에서 별도 논의 예정

### 4-2. 분신

- 발동 시점의 원본 로스터 볼을 각각 한 번씩 복사
- 기존 분신은 복사 대상에서 제외하여 분신이 다시 분신을 만들지 않음
- 복사본은 초기 볼 발사 간격과 같은 간격으로 순차 발사
- 각 복사본은 실제 발사 시점의 현재 조준 궤도를 사용
- 복사본도 Ground 충돌 후 발사 지점으로 회수되고 FIFO 큐를 통해 재발사
- 두 번째 회수 시 발사 지점에서 소멸하며 더 이상 재발사하지 않음
- 원본 볼에 연결된 볼 스킬을 복사본에도 동일하게 적용
- 활성 중 캐릭터 이펙트는 에셋 제작 단계에서 별도 논의 예정

---

## 5. UI

### 5-1. 배치

- 위치: 인게임 화면 하단 여백 (캐릭터 HP바 아래)
- 구성: 스킬 버튼 2개를 가로로 배치

### 5-2. 쿨타임 표현

- 쿨타임 진행 중: 버튼 위에 반투명 오버레이를 Image fillAmount Clockwise 방식으로 표시
- 오버레이 중앙에 남은 쿨타임 시간(초 단위) 텍스트 표시
- 쿨타임 종료 시: 오버레이 제거, 버튼 활성화
