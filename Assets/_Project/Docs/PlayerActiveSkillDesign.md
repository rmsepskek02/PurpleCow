# PlayerActiveSkill 기획 문서

이 문서는 인게임 플레이어 액티브 스킬 시스템의 기획 내용을 정리합니다.
기존 로그라이크 스킬 선택 시스템(볼/패시브 스킬)과는 독립된 시스템이며,
플레이어가 버튼을 직접 눌러 발동하는 4종의 스킬로 구성됩니다.
현재 버전은 프로토타입이므로 스킬 선택 없이 게임 시작 시 4개를 고정 지급합니다.

---

## 1. 개요

| 항목 | 내용 |
|------|------|
| 시스템 성격 | 기존 볼/패시브 스킬 선택 시스템과 별개 |
| 발동 방식 | 인게임 UI 버튼을 플레이어가 직접 탭하여 발동 |
| 스킬 지급 방식 | 프로토타입 기준: 게임 시작 시 4종 전부 고정 지급 |
| 수치 관리 | 모든 수치는 ScriptableObject로 Inspector에서 수정 가능 |

---

## 2. 스킬 목록

| 스킬명 | 쿨타임 | 효과 | 수치 |
|--------|--------|------|------|
| 필드 동결 | 30초 | 씬 내 모든 몬스터의 이동 및 행동 정지 | 지속 4초 |
| 버서크 | 30초 | 현재 활성화된 모든 볼 속도 증가 | 속도 1.5배, 지속 6초 |
| 분신 | 30초 | 현재 활성화된 모든 볼을 복사하여 추가 발사 | 2회 발사 후 복사본 소멸 |
| 마법 폭격 | 30초 | 씬 내 모든 몬스터에게 즉시 고정 피해 | 30 고정 피해 |

---

## 3. 공통 규칙

- 게임 시작 시 4개 모두 쿨타임 상태로 시작 (30초)
- 쿨타임 시작 시점: 스킬 발동 즉시
- 볼이 비행 중인 상태에서도 언제든 사용 가능

---

## 4. 스킬 상세

### 4-1. 필드 동결

- 씬 내 현재 살아있는 모든 몬스터의 이동 및 행동을 정지시킨다
- 지속 시간 4초 후 정상 상태로 복귀
- 시각 이펙트는 에셋 제작 단계에서 별도 논의 예정

### 4-2. 버서크

- 발동 시점에 씬 내 활성화된 모든 볼의 이동 속도를 1.5배로 증가
- 6초 동안 유지 후 원래 속도로 복귀
- 활성 중 캐릭터 이펙트는 에셋 제작 단계에서 별도 논의 예정

### 4-3. 분신

- 발동 시점에 씬 내 활성화된 모든 볼을 복사하여 동일한 수의 복사본을 추가 발사
- 복사본은 원본 볼과 동일한 속도·방향으로 발사됨
- 복사본은 2회 발사 후 소멸 (원본 볼은 유지)
- 보유 볼 수가 많을수록 발사되는 복사본 수도 많아져 효과가 강해짐
- 활성 중 캐릭터 이펙트는 에셋 제작 단계에서 별도 논의 예정

### 4-4. 마법 폭격

- 씬 내 현재 살아있는 모든 몬스터에게 즉시 30 고정 피해를 적용
- 데미지 계산에 별도 스케일링 없이 고정값 적용
- 시각 이펙트는 에셋 제작 단계에서 별도 논의 예정

---

## 5. UI

### 5-1. 배치

- 위치: 인게임 화면 하단 여백 (캐릭터 HP바 아래)
- 구성: 스킬 버튼 4개를 가로로 배치

### 5-2. 쿨타임 표현

- 쿨타임 진행 중: 버튼 위에 반투명 오버레이를 Image fillAmount Clockwise 방식으로 표시
- 오버레이 중앙에 남은 쿨타임 시간(초 단위) 텍스트 표시
- 쿨타임 종료 시: 오버레이 제거, 버튼 활성화

---

## 6. 미정 사항

아래 항목은 에셋 제작 단계에서 별도 논의 후 확정합니다.

| 항목 | 대상 스킬 |
|------|-----------|
| 발동 시 시각 이펙트 | 필드 동결, 마법 폭격 |
| 활성 중 캐릭터 이펙트 | 버서크, 분신 |
