# Design Agent 작업 기록

## 2026-07-05

### 작업 내용
플레이어 액티브 스킬 버튼 아이콘(필드 동결, 버서크, 분신 3종) 생성용 GPT 이미지 프롬프트 작성. 마법 폭격 아이콘은 이번 범위에서 제외.

### 결과
- 참고 이미지 `Assets/_Project/Docs/targetUI/KakaoTalk_20260703_115951058.jpg` (원본 게임 레벨업/스킬 선택 화면) 확인
  - 상단 스킬 슬롯 아이콘(불꽃 구슬, 고스트 오브, 수류탄, 시안 레이저 볼)이 아이콘 아트 스타일 레퍼런스
  - 어두운 라운드 사각/뱃지 프레임 + 선명한 심볼 + 부드러운 페인팅풍 셰이딩 + 은은한 글로우가 핵심 특징
- 3개 스킬 각각에 대해 영어 프롬프트 3종 작성 완료 (응답 텍스트로 전달, 파일 미생성)
  - 필드 동결: 시안/하늘색, 얼어붙은 시계/서리 결정 모티프
  - 버서크: 빨강/주황, 주먹+불꽃/속도선 모티프
  - 분신: 보라/자주, 겹쳐진 반투명 고스트 오브(원본+잔상) 모티프

### 주요 결정사항
- 심볼만이 아니라 뱃지 프레임까지 포함해서 생성하도록 프롬프트에 명시 (참고 이미지 스타일과 일치)
- 모든 프롬프트에 공통 구조 적용: 아트 스타일 → 프레임 → 심볼 모티프 → 컬러 팔레트 → 렌더링 디테일 → 포맷(정사각형/배경 처리) 순서로 통일하여 일관성 유지
- 프레임 배경은 스킬 컬러에 맞춰 톤온톤 다크(예: 동결=다크 네이비, 버서크=다크 마룬, 분신=다크 바이올렛)로 미세하게 구분 지정 — 완전히 동일한 검정보다 스킬 정체성이 살짝 드러나도록 (1차 버전, 이후 피드백으로 수정됨)

### 1차 수정 (같은 날, 사용자 피드백 반영)
- 프레임을 스킬별로 다르게 하지 말고 3개 모두 동일한 라운드 사각형 빨간 프레임으로 통일
- 정사각형 1:1 비율은 유지 및 강조
- 필드 동결/버서크는 심볼 컨셉 그대로 유지, 프레임만 빨간색으로 교체
- 분신은 심볼을 "겹쳐진 반투명 고스트 오브 2개"에서 "보라/자주색 볼 여러 개가 튕기며 난무하는 형태"로 전면 교체 (컬러 팔레트는 보라/자주 유지)
- `Assets/_Project/Docs/targetUI/KakaoTalk_20260705_171750210*.jpg` (빨간 후드 캐릭터, 인게임 플레이 화면)를 재참고해 전체 아트 톤을 "무겁고 어두운 다크 판타지"에서 "캐주얼하고 귀여운(cute, casual) 모바일 게임" 톤으로 전환 — "dark near-black gradient" 등 무거운 표현 제거, 밝고 경쾌한 렌더링 지시로 교체
